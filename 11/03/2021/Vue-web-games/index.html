Vue2 Web Games

Link of the Web

  • Hosted on Firebase

Purpose of the Project

  • To learn how to setup Vue.js, Vuex, and webpack.
  • To apply Vue.js Component options and Directives
  • To implement algorithms of the games with Vue.js components.
  • To laern how to manage Vuex store.
  • To learn VueRouter to implement sub-pages in the SPA app.
  • To learn dynamic route matching
  • To learn parent-child components structure and its communication
