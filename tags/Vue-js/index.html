Vue.js Introduction


Vue Directives & Component options

Vue Official Web

Component options

  • Data

    • data
       Type Object, accepts function, and returns an object with properties.
    • props
       Type Array, accepts data from the parent component. Contains a set of attributes.
    • methods (event handling)
       Type Object of Function.
    • computed (computed properties)
       Type Object of Function (collections).
    • watch
       Key name is the attribute/expression to watch and its value is the callback.

    => Be careful with arrow functions for methods, computed and watch.

  • DOM

    • el
       Type CSS string or Html Element. Can mount on a DOM element when new is used for instance creation. ‘#’ prefix is used in value to get the id of an Element.
    • template
       Type string. This is used as the markup(html) for the Component/instance. Template replace the mounted element.

Directives

  • v-if | v-else | v-else-if
     Display element(s) by conditions initially contained in v-if, then additionally add paths with v-else or v-else-if
  • v-for
     Has ability to display multiple elements with source data. Can use Array, Object, string etc. Need to specify an item in Array/Object with a index/key in a group to render. If reorder is required, put a key value in ‘:key’ value after v-for.
  • v-on:
     Has shorthand ‘@’. Work with Function or Object, to handle event(s). Has modifiers to use (@event .MODIFIER).
  • v-bind
     Has shorthand ‘:’. Work with Any (argument: attr or prop) or Object (no arg.), to handle dynamic binding of attr-expr or prop-expr. Class binding is to bind style, Prop binding is to bind with child prop which should be declread properly in the child component. Has modifiers to use (@event .MODIFIER).
  • v-model
     For input|select|textare elements or components. Generates two-way binding on them. Has modifiers to use (@event .MODIFIER).
